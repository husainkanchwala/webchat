webchat
========

chat app using node.js, express.js, socket.io and redis as database

first install 
    
    nodejs

then, install modules

    npm install -d

also you will require 
  
    jquery.cookie.js
    
In order to run it first install redis database and then run server.js using

	redis-server

	node server.js
